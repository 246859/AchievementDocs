# 玩家数据

TIP

建议先了解下玩家数据的JSON结构，玩家数据

# 获取玩家的成就对象

getPlayerInfo(xuid)
1
  • 返回值：Object - 具体结构可以查看Data.json文件
  • 参数：
    • xuid:string - 玩家的xuid

# 获取玩家指定类型的成就信息

getPlayerTypeInfo(xuid, type)
1
  • 返回值： Object - 玩家成就类型对象
  • 参数：
    • xuid: string - 玩家xuid
    • type: string - 成就类型

# 获取玩家指定类型中指定成就词条的信息

getPlayerKeyInfo(xuid, type, key)
1
  • 返回值：Object - 玩家成就词条对象
  • 参数：
    • xuid: string - 玩家xuid
    • type: string - 成就类型
    • key: string - 词条key

# 获取玩家成就类型完成情况的统计

getPlayerTypeStatistic(xuid)
1
  • 返回值：{unFinished: Integer,finished: Integer,unFinishedList: *[], finishedList: *[]}
  • 参数：
    • xuid - 玩家xuid

# 获取玩家指定成就类型中成就完成情况统计

getPlayerDetailStatistic(xuid, type)
1
  • 返回值：{unFinishedList: *[], finishedList: *[]}
  • 参数：
    • xuid: string - 玩家xuid
    • type: string - 成就类型
