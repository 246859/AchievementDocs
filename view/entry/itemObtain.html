# 物品成就

成就描述

  • 名称：物品成就
  • 类型名：itemObtain
  • 默认开启：true
  • json：
{
	"itemObtain": {
		"enable": true,
		"name": "物品成就",
		"details": {},
		"regx": {}
	}
}
