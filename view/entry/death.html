# 死亡成就

当玩家死亡时，便会触发此种类型的成就。

  • 名称：死亡成就
  • 类型名：death
  • 默认开启：true
  • 传入值：实体的标准类型名
  • 词条数量：30
  • json：
{
	"death": {
		"enable": true,
		"name": "死亡成就",
		"details": {},
		"regx": {}
	}
}
