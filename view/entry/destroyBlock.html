# 挖掘成就

成就描述

  • 名称：挖掘成就
  • 类型名：destroyBlock
  • 默认开启：true
  • json：
{
	"destroyBlock": {
		"enable": true,
		"name": "挖掘成就",
		"details": {},
		"regx": {
			"minecraft:log": "minecraft:log"
		}
	}
}
